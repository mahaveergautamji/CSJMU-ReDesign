# CSJMU-ReDesign

A modern, reimagined version of the Chhatrapati Shahu Ji Maharaj University (CSJMU), Kanpur website.

---

## Purpose

This project aims to present a fresh, user-friendly, and visually appealing redesign of CSJMU's official website. The goal is to deliver a seamless and intuitive experience for students, faculty, and visitors, blending modern UI trends with real-world features of a campus portal.

---

## Features

- **Modern Home Page**  
  A visually engaging homepage highlighting the university’s vision, quick links to important resources, notices, featured events, and leadership team—all designed with contemporary UI/UX best practices.

- **Responsive Design**  
  Built with [Tailwind CSS](https://tailwindcss.com/) and Google Fonts, fully responsive for use on desktops, tablets, and mobile devices.

- **Student and Faculty Portal**  
  Interactive login pages allow students and faculty to enter a dedicated portal:
  - **Student Portal:**  
    - Personalized dashboard with today’s schedule, subject-wise attendance, assignments, and quizzes.
    - Class schedule with holiday calendar.
    - Notices and circulars.

  - **Faculty Portal:**  
    - Tools to mark attendance.
    - View attendance history for students.
    - Manage assignments and quizzes.

- **Dynamic Timetable & Attendance**  
  Hardcoded but realistic datasets for students, faculty, class schedules, labs, holidays, and mock attendance functionality, simulating a real-world student/faculty portal.

- **Interactive UI Components**  
  - Flip cards, glassmorphism, neon effects, and confetti animation for holidays.
  - Mobile-friendly sidebar, day selectors, and modals.

---

## What’s Inside

- **HTML, CSS (Tailwind), and Vanilla JavaScript**  
  All website logic, data, and UI are implemented in a single `index.html` file―no frameworks or backend required.

- **Self-Contained Data**  
  - Student and faculty lists for different branches (sample data).
  - Holiday lists and demo timetable.
  - Mock data for assignments, attendance, and notices.

- **Easy Customization**  
  Add your own data by editing JavaScript objects within `index.html`.

---

## Getting Started

Simply clone or download the repository and open `index.html` in any modern web browser.

```bash
git clone https://github.com/mahaveergautamji/CSJMU-ReDesign.git
cd CSJMU-ReDesign
# Open index.html in your browser
```

_No additional setup required!_
